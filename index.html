Mozilla เปิดตัว Firefox Reality เว็บเบราว์เซอร์ออกแบบใหม่พัฒนาเพื่อใช้กับ VR, AR

Mozilla เปิดตัว Firefox Reality เว็บเบราว์เซอร์ใหม่ที่ถูกออกแบบเพื่อการใช้งานกับ VR และ AR โดยเฉพาะเบราว์เซอร์ Firefox Reality นี้ถูกพัฒนาขึ้นโดยใช้เทคโนโลยีเว็บของ Firefox ปัจจุบัน และปรับปรุงด้วย Servo เว็บเอนจินที่กำลังอยู่ในระหว่างการทดลองของ Mozilla ซึ่งภายในสัปดาห์หน้า Firefox จะเริ่มปล่อยข้อมูลต่าง ๆ เช่น กระบวนการออกแบบ, โชว์การทดสอบกับ headset หลายรุ่น, ทดสอบการใช้งาน computer vision pipeline ด้วยการใช้ WebAssembly, อินทิเกรต Servo พร้อมกับส่วนขยายที่กำลังอยู่ระหว่างการทดลองเข้ากับ API กราฟิกของ WebGL


Firefox Reality ออกแบบมาให้เหมาะกับการใช้งานบนเฮดเซ็ต VR, AR หลาย ๆ รูปแบบ แต่ในช่วงเริ่มต้นของการพัฒนานั้น ซอร์สโค้ดจะรันบน Daydream และ GearVR เท่านั้น และตอนนี้ Mozilla ก็ได้ปล่อยซอร์สโค้ดให้นักพัฒนานำไป build เพื่อใช้งานกับแพลตฟอร์มอื่นได้ สามารถดาวน์โหลดซอร์สโค้ดได้ที่ GitHub


Mozilla จะคอยอัพเดตเรื่องราวต่าง ๆ ของ Firefox Reality สามารถติดตามได้ผ่านทาง Mozilla Mixed Reality Blog และ Twitter @mozillareality
